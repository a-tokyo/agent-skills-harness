# Documentation

The Agent Skills Harness is a factory for building production-grade agent skills via a five-phase pipeline
(interview → research → draft → autoresearch → verify). These pages are rendered directly from the repo's
canonical docs.

- [**I/O Contract**](/docs/io-contract) — what you provide and what you get back. **Start here.**
- [**Workspace Layout**](/docs/workspace-layout) — the `builds/<skill>/{input,work,output}` layout, file by file.
- [**Usage Guide**](/docs/usage-guide) — end-to-end walkthrough of a build.
- [**Architecture**](/docs/architecture) — design overview and how the pieces fit.
- [**Rubric Format**](/docs/rubric-format) — how evaluation rubrics are defined.
- [**METRIC Protocol**](/docs/metric-protocol) — the `METRIC name=value` measurement line format.
- [**Benchmark**](/docs/benchmark) — a blind, panel-judged factory build vs a human reference.
- [**Skill-Authoring Best Practices**](/docs/skill-authoring-best-practices) — the official rules the factory follows.

Everything here lives in the [GitHub repository](https://github.com/a-tokyo/agent-skills-harness); the factory
skill is also [published standalone](https://github.com/a-tokyo/agent-skills) for `npx skills`.
